### 2. 생성된 파일

#### 2.1 extract_pdf_text.py

**주요 기능:**
- PDF 파일에서 텍스트 추출
- 폴더 내 모든 PDF 파일 일괄 처리
- 추출된 텍스트를 개별 텍스트 파일로 저장

**주요 함수:**

1. **`extract_text_from_pdf(pdf_path)`**
   - 단일 PDF 파일에서 텍스트 추출
   - 페이지별로 구분하여 텍스트 반환
   - 오류 처리 포함

2. **`extract_all_pdfs(pdf_folder, output_folder=None)`**
   - 지정된 폴더의 모든 PDF 파일 처리
   - 자동으로 출력 폴더 생성 (`repository/extracted_texts`)
   - 진행 상황 콘솔 출력

**기술 스택:**
- Python 3.x
- `pypdf` 라이브러리 (PDF 처리)
- `pathlib` (경로 처리)

#### 2.2 requirements.txt

```
pypdf>=3.17.0
```

**설명:**
- `pypdf` 3.17.0 이상 버전 필요
- PDF 파일 읽기 및 텍스트 추출 기능 제공

#### 2.3 README.md

프로젝트 사용 설명서 포함:
- 설치 방법
- 사용 방법
- 기능 설명

### 3. 프로그램 특징

1. **자동화**
   - 폴더 내 모든 PDF 파일 자동 탐지 및 처리
   - 출력 폴더 자동 생성

2. **사용자 친화적**
   - 진행 상황 실시간 출력
   - 페이지 수 표시
   - 오류 메시지 명확히 표시

3. **구조화된 출력**
   - 각 페이지별로 구분된 텍스트
   - 원본 파일명 유지 (확장자만 .txt로 변경)

4. **오류 처리**
   - 파일 읽기 오류 처리
   - 폴더 없음 오류 처리
   - 예외 상황 안전하게 처리

### 4. 사용 방법

#### 4.1 설치

```bash
pip install -r requirements.txt
```

#### 4.2 실행

```bash
python extract_pdf_text.py
```

#### 4.3 출력

- 추출된 텍스트는 `repository/extracted_texts/` 폴더에 저장됩니다.
- 각 PDF 파일마다 하나의 `.txt` 파일이 생성됩니다.
- 파일명 형식: `{원본PDF파일명}.txt`

### 6. 코드 구조

```python
# 주요 흐름
```
